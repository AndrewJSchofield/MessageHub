# 使用 Kafka REST API
{: #rest_using}

Kafka REST API 提供了一个用于访问 Kafka 集群的 RESTful 界面。您可以使用该 API 轻松生成和使用消息以及完成管理任务。有关参考文档，请参阅 [Confluent Platform 文档 ![外部链接图标](../../icons/launch-glyph.svg "外部链接图标")](http://docs.confluent.io/2.0.0/){:new_window}。

如果使用的是 CURL，那么可以通过将以下行添加到命令行，调整 [Confluent 文档 ![外部链接图标](../../icons/launch-glyph.svg "外部链接图标")](http://docs.confluent.io/2.0.0/){:new_window} 中详细描述的代码示例：
<pre class="pre">-H "X-Auth-Token: <var class="keyword varname">APIKEY</var>"</pre>
